# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in daft-h3, please report it privately via [GitHub's security advisory feature](https://github.com/gweaverbiodev/daft-h3/security/advisories/new).

Please do not open a public issue for security vulnerabilities.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |
